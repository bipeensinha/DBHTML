Structure Tags

The fundamental tags that define the structure of an HTML document:

  • <html> - Root element
  • <head> - Container for metadata
  • <title> - Title of the document
  • <body> - Body of the document
  • <header> - Introductory content
  • <nav> - Navigation links
  • <section> - Section of a document
  • <article> - Independent content
  • <aside> - Tangentially related content
  • <footer> - Footer content

Formatting Tags

Tags used for formatting and structuring text:

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <br> - Line break
  • <hr> - Horizontal rule
  • <strong> - Bold text
  • <em> - Italic text
  • <a> - Anchor (link)
  • <ul> - Unordered list
  • <ol> - Ordered list
  • <li> - List item
  • <table> - Table
  • <tr> - Table row
  • <th> - Table header
  • <td> - Table data cell

Multimedia Tags

Tags for embedding multimedia content:

  • <img> - Image
  • <audio> - Audio content
  • <video> - Video content
  • <source> - Media source for <audio> and <video>
  • <iframe> - Inline frame

Form Tags

Tags used to create forms and collect user input:

  • <form> - Form element
  • <input> - Input field
  • <label> - Label for input
  • <textarea> - Multi-line text input
  • <button> - Button
  • <select> - Dropdown list
  • <option> - Option in a dropdown
